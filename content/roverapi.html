Rover API DocumentationÂ¶

The Rover API is built with the Roverapp software and allows to control the different components of the Rover such as reading sensor values or steering the wheels. Furthermore, it features Cloud communication via a built-in MQTT-client based on Eclipse Paho.

Note

Rover API Documentation is provided in https://app4mc-rover.github.io/rover-app.
