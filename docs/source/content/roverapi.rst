Rover API Documentation
#########################

.. image:: ../roverstatic/images/roverapi_logo.png
   :width: 40%
   :align: center
   :alt: ../roverstatic/images/roverapi_logo.png

The Rover API is built with the Roverapp software and allows to control the different components of the Rover such as reading sensor values or steering the wheels.
Furthermore, it features Cloud communication via a built-in MQTT-client based on Eclipse Paho.

.. note:: Rover API Documentation is provided in `https://app4mc-rover.github.io/rover-app <https://app4mc-rover.github.io/rover-app>`_.
